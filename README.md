# TODO アプリ

## 各種情報

### React のバージョン

v17.0.2

### Node のバージョン

v14.17.6

### 使用しているモジュール

- axios
- json-server (DEV)
- ulid

## 環境構築

### ローカル用DBのJSONファイルを作成
```
cp db.local.json db.json
```

### node モジュールをインストール

```
yarn
```

### アプリ起動

```
yarn start
```

### モックサーバー起動

```
yarn mock:server
```
